prepend

Source:
  • prepend.js, line 3

jQuery equivalent: $.prepend

Inserts elements as first child of the current selection. Returns the new elements as a D3 selection.

selection.prepend(tagName);

Example

d3.selectAll('li')
 .prepend('a')
 .text('Some Link')
 //do somethin else with the link
